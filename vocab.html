小学阶段学的多是具体名词和基础动词；初中骤然要求掌握抽象词、多义词、构词复杂词。孩子用原来「反复抄写、死记释义」的方法效率极低——研究表明，单词在不同语境中出现 6-7 次才能真正记住。

STEP 01
诊断缺口，生成专属词单
不要用通用词书，按教材版本让 AI 定制词单
核心逻辑：每个孩子的缺口不一样，教材版本不同词序也不同。让 AI 直接生成定制词单，比买一本词书高效得多。
Prompt A生成初一高频词单（按教材版本）
适用场景：开学前一个月，快速锁定需要补的词。Claude / ChatGPT 均可使用。
// PROMPT · 点击右上角复制
你是初中英语教学专家。请完成以下任务： 【任务】列出【人教版】初一上册中，小学毕业生最容易陌生的50个核心词。 【输出格式】请按以下格式输出每个词： 单词 | 词性 | 中文释义 | 一个生活场景例句（不超过15词）| 1个同根词 【额外要求】 - 优先选择：多义词、抽象词、易混淆词 - 例句场景请贴近初中生日常生活 - 最后单独列出3个「最容易考的」词并注明原因 开始输出：
替换「人教版」为实际版本外研社 / 译林 / 北师大均可

STEP 02
语境记忆：每词3个场景句
不背释义背用法，记住怎么用比记住什么意思更重要
研究表明：单词在不同语境中出现 6-7 次才能真正记住。传统背词书每个词只见一次，所以忘得快。AI 可以无限生成新场景句，这是它最大的优势。
Prompt D生成生活场景句（单词精讲）
针对单个词，生成3个不同生活场景的例句，帮孩子在具体情境中记住用法。
// PROMPT · 点击右上角复制
请帮我用「convenient」这个词，造3个不同场景的英语句子： 场景要求： ① 场景一：放学回家，妈妈问路线 ② 场景二：周末去超市购物 ③ 场景三：用手机叫外卖 格式： - 英文句子（不超过15词，初一水平） - 中文翻译 - 用中文说明这个词在这句话里的具体含义 最后加一句：这个词最容易和哪个词混淆？怎么区分？ 【注意】：把「convenient」替换为你要学的单词即可复用。
替换单词即可复用贴近生活含辨析

STEP 03
AI 出题 + 即时纠错
针对弱点无限出新题，错了立刻知道为什么
AI 出题的核心优势：针对孩子的错词定向出题，永远不重复，且做完即反馈。传统习题册做完就没有了，AI 可以无限续杯。
Prompt G完形填空出题（单句版）
最常见的题型，适合每日10分钟快速练习，AI 出题后等孩子回答再给答案和解析。
// PROMPT · 点击右上角复制
请用以下10个词出5道填空题，并严格按我说的步骤执行： 【词单】：achieve, environment, suggest, although, probably, decision, introduce, necessary, improve, polite 【步骤】 第一步：出5道单句填空题（每题空一个词，题目用"___"表示） 第二步：等待我的回答（我会在下一条消息输入答案） 第三步：告诉我哪道题错了，并逐题解释： - 正确答案是什么 - 为什么选这个词 - 这个词在这里是什么词性、什么意思 - 给一个额外的例句帮助记忆 【注意】：请现在只输出第一步的5道题，不要提前给答案。
分步骤执行即时纠错含解析

STEP 04
构词规律：一个词变十个词
学会前缀/后缀规律，遇到新词能推断，才是真正的提速
初中词汇有清晰的构词规律。让 AI 整理词族树，孩子不用死记每个新词，而是通过规律推断——这是词汇量快速增长的杠杆。

PLAN
6周暑假补词计划
每天 15-20 分钟，家长陪学路线图
不需要每天很长时间，关键是坚持。每天15-20分钟，6周下来可以稳固掌握500-600个新词，覆盖初一上学期大部分词汇需求。
第 1-2 周
词单 + 语境句
每天20个新词。用 Prompt A 生成词单，Prompt D 造场景句，亲子朗读。以「看到能认出」为目标，不要求默写。
第 3-4 周
AI 出题 + 纠错
每天复习旧词 + 做10道 Prompt G 的填空题。遇到错词记下，用 Prompt H 出专项题再巩固。
第 5 周
构词规律系统学
用 Prompt J/K 学前缀后缀，每天1个规律。配合 Prompt L 生成速查表贴在书桌旁。
第 6 周
综合复习 + 阅读
用 Prompt B 再做一次诊断，找剩余漏洞。同时开始接触简单英语阅读材料。
